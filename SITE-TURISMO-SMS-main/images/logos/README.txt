============================================================
LOGOS INSTITUCIONAIS - INSTRUÇÕES
============================================================

Coloque os logos das instituições parceiras nesta pasta.

Arquivos esperados:
- governo-pr.png (Governo do Estado do Paraná)
- setu.png (Secretaria de Estado do Turismo)
- viaje-parana.png (Viaje Paraná - Turismo)
- igr-sul.png (IGR Sul do Paraná / ATEMA)
- ig-mathe.png (Selo IG São Matheus - Erva-mate)
- prefeitura.png (Prefeitura Municipal) [já existe]

Especificações recomendadas:
- Formato: PNG com fundo transparente
- Tamanho: 150x80 pixels (ou proporcional)
- Resolução: 72-150 dpi

Após adicionar os logos, edite o arquivo index.html:
1. Localize a seção "RÉGUA DE LOGOS INSTITUCIONAIS"
2. Substitua cada div.logo-placeholder pelo <img> correspondente
3. Exemplo: 
   DE:  <div class="logo-placeholder">...</div>
   PARA: <img src="images/logos/governo-pr.png" alt="Governo do Paraná">
